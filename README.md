# ImageProcessingManualFunction

A Python Tkinter project for **manual image processing**.  
Implements basic operations, point operations, histogram processing, linear & non-linear filters, noise addition, morphology, and dithering **pixel by pixel** without using built-in image processing functions.

## Requirements
- Python 3.x
- numpy
- matplotlib
- scikit-image
- tkinter
